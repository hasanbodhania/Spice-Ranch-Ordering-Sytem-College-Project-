<h2>ğŸ§  Program Structure</h2>
<p><strong>Globals.vb</strong> is used to store shared data that needs to be accessed across multiple forms. This includes:</p>
<ul>
    <li>Arrays for selected menu items</li>
    <li>Quantities</li>
    <li>Prices</li>
    <li>Running totals</li>
</ul>
<p>Using <strong>Globals.vb</strong> allows:</p>
<ul>
    <li>Data to persist between pages</li>
    <li>Cleaner form-to-form communication</li>
    <li>Avoiding duplication of logic</li>
</ul>
